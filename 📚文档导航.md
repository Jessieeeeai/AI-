## 🛠️ 部署脚本

### 主应用部署 → **[deploy_runpod.sh](./deploy_runpod.sh)**
```bash
# 用法
bash deploy_runpod.sh
```
- **功能**: 部署VideoAI Pro主应用
- **时间**: 5-10分钟
- **包含**: Node.js, Redis, FFmpeg, PM2, Nginx

### AI服务部署 → **[deploy_ai_services.sh](./deploy_ai_services.sh)**
```bash
# 用法

```
- **功能**: 部署IndexTTS2 + ComfyUI
- **时间**: 30-60分钟
- **包含**: Python环境, GPU服务, 模型下载

## 📞 获取帮助

### 查看日志
```bash
pm2 logs                    # 查看所有日志
pm2 logs videoai            # 查看后端日志
pm2 logs indextts2          # 查看TTS日志
pm2 logs comfyui            # 查看视频生成日志
```

### 常用检查
```bash
pm2 status                  # 服务状态
nvidia-smi                  # GPU状态
df -h                       # 磁盘空间
free -h                     # 内存使用
systemctl status nginx      # Nginx状态
redis-cli ping              # Redis状态
```

### 文档内搜索
```bash
# 在文档中搜索关键词
grep -r "关键词" *.md

# 例如：搜索"GPU"
grep -r "GPU" *.md
```
